常见问题

以下列出用户在使用 C++ API 各个部分时可能遇到的一些常见问题。

C++扩展

PyTorch/ATen 中的未定义符号错误

问题：您导入扩展后，出现一个 ImportError 错误，表示 PyTorch 或 ATen 中的某些 C++符号未定义。例如：

>>> import extension
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
ImportError: /home/user/.pyenv/versions/3.7.1/lib/python3.7/site-packages/extension.cpython-37m-x86_64-linux-gnu.so: undefined symbol: _ZN2at19UndefinedTensorImpl10_singletonE

修复：在导入您的扩展之前，请使用 import torch 进行修复。这将使您的扩展所依赖的 PyTorch 动态（共享）库中的符号可用，以便在导入扩展后进行解析。

我使用 at:: 中的函数创建了一个张量，并遇到了错误

问题：您使用例如 at::ones 或 at::randn 或任何其他来自 at:: 命名空间的张量工厂创建了一个张量，并遇到了错误。

修复：在工厂函数调用中用 torch:: 替换 at:: 。您永远不应该使用来自 at:: 命名空间的工厂函数，因为它们将创建张量。相应的 torch:: 函数将创建变量，您应该只处理代码中的变量。
